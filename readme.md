# Electron Shopping List Desktop App

* Install all packages using 'npm install'
* To start  -  npm start
* To package -  </br>
        - Windows - npm run package-win </br>
        - Mac - npm run package-mac  </br>
        - Linux - npm run package-linux  </br>  </br>
